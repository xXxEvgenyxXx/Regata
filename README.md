# Regata.team - Корпоративные регаты и тимбилдинг на море

Веб-сайт для компании Regata.team, специализирующейся на организации корпоративных регат и тимбилдинговых мероприятий на море.

## Описание проекта

Сайт создан на основе дизайна из Figma и включает в себя:

- **Навигация** - фиксированная навигационная панель с логотипом, меню и контактным телефоном
- **Hero секция** - главный экран с призывом к действию
- **О компании** - информация о компании и её услугах
- **Команда** - представление членов команды
- **Контакты** - форма обратной связи и контактная информация

## Технологии

- HTML5
- CSS3
- Адаптивный дизайн
- Шрифт Montserrat (Google Fonts)
- SVG иконки

## Особенности

- **Адаптивный дизайн** - сайт корректно отображается на всех устройствах
- **Современный UI/UX** - использование современных принципов дизайна
- **Оптимизация** - легкий и быстрый сайт
- **Доступность** - семантическая разметка и удобная навигация

## Установка и запуск

1. Скачайте все файлы проекта
2. Откройте `index.html` в любом современном браузере
3. Для разработки рекомендуется использовать локальный сервер

## Адаптивность

Сайт адаптирован для следующих разрешений:
- **Desktop** (1200px+) - полная версия с горизонтальным меню
- **Tablet** (768px - 1199px) - адаптированная версия
- **Mobile** (до 767px) - мобильная версия с гамбургер-меню

### Особенности мобильной версии:
- **Гамбургер-меню** - скрытое меню с анимацией
- **Оптимизированные размеры** - все элементы адаптированы под мобильные экраны
- **Улучшенная навигация** - удобное управление на сенсорных экранах
- **Адаптивные отступы** - оптимальные размеры для мобильных устройств
